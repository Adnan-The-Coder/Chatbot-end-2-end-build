Integration Examples

cURL
curl -X POST "http://localhost:8080/chat/datanyx" \ -H "Content-Type: application/json" \ -d '{"message": "What services do you offer?"}'
Python
import requests response = requests.post( "http://localhost:8080/chat/datanyx", json={"message": "What services do you offer?"} ) data = response.json() print(f"Chatbot: {data['response']}")
JavaScript / Node.js
const response = await fetch('http://localhost:8080/chat/datanyx', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ message: 'What services do you offer?' }) }); const data = await response.json(); console.log(data.response);

Quick Deploy

Local Development
git clone https://github.com/Adnan-The-Coder/Chatbot-end-2-end-build.git cd Chatbot-end-2-end-build pip install -r requirements.txt uvicorn api:app --reload --host 0.0.0.0 --port 8080
Docker Deployment
docker build -t ml-chatbot-api:latest . docker run -d -p 8080:8000 --name chatbot ml-chatbot-api:latest
Docker Compose (Production)
version: '3.8' services: chatbot-api: build: . ports: - "8080:8000" environment: - ENV=production restart: unless-stopped
